## 三、新增资金加仓策略

### 定期加仓（每月/每季新增资金）

**操作流程**：

1. **计算当前持仓比例**
   - 当前比例 = 单只基金市值 ÷ 组合总市值

2. **确定加仓方式**
   - 情况A：各资产比例接近目标（偏差<3%）→ 按目标比例加仓
   - 情况B：有资产明显低配（偏差>3%）→ 优先加仓低配资产

3. **执行加仓**
   - 按计算出的金额申购对应基金
   - 建议在每月固定日期（如发薪日）执行

**计算公式**：

```
单只基金加仓金额 = 新增资金 × 目标比例
或
单只基金加仓金额 = 新增资金 × (目标比例 - 当前比例) [优先补低配时]
```

### 大额资金加仓（如年终奖）

**操作建议**：

1. **分2-3个月投入**，平滑市场波动风险
2. **优先补低配资产**，加速恢复目标比例
3. **记录加仓成本**，方便未来税务计算

## 四、紧急用钱减仓策略

### 减仓优先级原则

1. **持有期限优先**：先赎回持有超过1年的份额（免赎回费）
2. **盈利优先**：先赎回有盈利的份额
3. **比例维持**：尽量按当前比例赎回，保持配置平衡

### 减仓操作步骤

1. **确定赎回总额**：根据资金需求确定
2. **计算各基金赎回金额**：

```
单只基金赎回金额 = 总赎回金额 × (该基金当前市值 ÷ 组合总市值)
```

3. **检查持有期限**：

- 优先从持有超过1年的基金中赎回
- 如果某基金持有不足1年份额不够，从其他持有超1年基金多赎

4. **执行赎回**：

- 在交易日下午3点前提交申请
- 注意资金到账时间（通常T+3工作日）

### 临时周转方案

**建议设立应急资金层**：

1. **第一层**：货币基金（1-2个月生活费）
2. **第二层**：短期债券基金（2-6个月生活费）
3. **第三层**：投资组合（长期资金）

## 五、再平衡策略（详细表格版）

| 再平衡类型         | 判断时间点                               | 触发条件                       | 具体判断标准                               | 操作示例（假设组合总市值100万）                                                                        |
| ------------------ | ---------------------------------------- | ------------------------------ | ------------------------------------------ | ------------------------------------------------------------------------------------------------------ |
| **定期再平衡**     | 每季度末<br>（3/6/9/12月最后一个交易日） | 时间固定触发                   | 无论当前比例如何，季度末统一检查执行       | 例：6月30日检查，发现沪深300当前24%（目标20%），则赎回4万元（100万×4%）沪深300，按低配比例申购其他基金 |
| **阈值再平衡**     | 每月第一个周末检查                       | 单一资产偏离目标±4%            | 当前比例 ≥ 目标比例+4%<br>或 ≤ 目标比例-4% | 例：国开债目标15%，当前19%（超过15%+4%），则触发再平衡，赎回4万元国开债，申购低配资产                  |
| **股票内部再平衡** | 与阈值检查同时                           | 沪深300:中证500比例偏离2:1 ±3% | (沪深300比例 ÷ 中证500比例) ≥ 2.3 或 ≤ 1.7 | 例：沪深300当前23%，中证500当前9%，比例2.56>2.3，需调整。赎回沪深3002万，申购中证5002万，恢复2:1       |
| **债券内部再平衡** | 与阈值检查同时                           | 国开债:国债比例偏离1:1 ±2%     | (国开债比例 ÷ 国债比例) ≥ 1.2 或 ≤ 0.8     | 例：国开债当前17%，国债当前13%，比例1.31>1.2，需调整。赎回国开债2万，申购国债2万，恢复1:1              |
| **极端情况处理**   | 任意时间发现                             | 任一资产偏离目标±8%            | 市场极端波动导致大幅偏离                   | 例：黄金目标10%，因暴涨至18%（超过10%+8%），立即赎回8万元黄金，按比例申购其他资产                      |

### 再平衡操作通用流程

1. **计算当前市值比例**：记录每个基金当前市值，计算占总资产比例
2. **识别偏离资产**：标记所有超出容忍范围（±4%）的资产
3. **计算调整金额**：

```
赎回金额 = 组合总市值 × (当前比例 - 目标比例) [当当前比例>目标时]
申购金额 = 组合总市值 × (目标比例 - 当前比例) [当当前比例<目标时]
```

4. **执行交易**：

- 赎回超配基金（优先赎回持有>1年份额）
- 申购低配基金（按目标比例分配申购金额）

5. **记录归档**：记录再平衡日期、原因、操作详情

## 六、费用优化建议

### 申购费优惠渠道

- 天天基金、蚂蚁财富：通常申购费1折
- 基金公司官网/APP：可能有更低费率
- 设置定投计划：部分平台免申购费

### 赎回费节省技巧

| 持有时间 | 赎回费率 | 优化建议       |
| -------- | -------- | -------------- |
| <7天     | 1.5%     | 避免短期赎回   |
| 7天-1年  | 0.5%     | 尽量持有超1年  |
| >1年     | 0%       | 优先赎回这部分 |

### 长期持有成本

- 预计年化总成本：0.35%-0.45%
- 主要构成：管理费+托管费+申购费分摊

## 八、重要提醒

### 投资前须知

1. **资金性质**：用3-5年不用的闲钱投资
2. **风险认知**：任何投资都有风险，历史表现不代表未来
3. **长期坚持**：全天候策略需坚持3年以上才能体现效果
4. **纪律执行**：严格按照策略执行，避免情绪化操作

### 特别注意事项

1. **国开债波动**：7-10年久期债券对利率敏感，4%阈值有助于及时控制风险
2. **QDII基金特性**：国泰大宗商品为QDII基金，净值公布较晚，注意申购赎回时间
3. **税务考虑**：国开债利息收入需缴税，通过基金投资由管理人处理税务
4. **应急准备**：始终保持3-6个月生活费的流动性资产，避免被迫在不利时机赎回投资组合

### 常见问题处理

1. **基金暂停申购**：用表格中的替代基金或暂时投入货币基金
2. **大额资金需求**：提前1-2个月开始分批赎回
3. **市场极端波动**：坚持再平衡纪律，这正是策略价值所在
4. **平台操作问题**：熟悉交易平台功能，设置价格提醒和定投计划

### 关键时间节点备忘

- **季度末**：定期再平衡检查
- **每月初**：阈值检查
- **每年1月**：年度复盘与策略评估
- **持有满1年**：检查是否有可免税赎回的份额
