## 三、新增资金加仓策略

### 定期加仓（每月/每季新增资金）

**操作流程**：

1. **计算当前持仓比例**
   - 当前比例 = 单只基金市值 ÷ 组合总市值

2. **确定加仓方式**
   - 情况A：各资产比例接近目标（偏差<3%）→ 按目标比例加仓
   - 情况B：有资产明显低配（偏差>3%）→ 优先加仓低配资产

3. **执行加仓**
   - 按计算出的金额申购对应基金
   - 建议在每月固定日期（如发薪日）执行

**计算公式**：

```
单只基金加仓金额 = 新增资金 × 目标比例
或
单只基金加仓金额 = 新增资金 × (目标比例 - 当前比例) [优先补低配时]
```

### 大额资金加仓（如年终奖）

**操作建议**：

1. **分2-3个月投入**，平滑市场波动风险
2. **优先补低配资产**，加速恢复目标比例
3. **记录加仓成本**，方便未来税务计算

## 四、紧急用钱减仓策略

### 减仓优先级原则

1. **持有期限优先**：先赎回持有超过1年的份额（免赎回费）
2. **盈利优先**：先赎回有盈利的份额
3. **比例维持**：尽量按当前比例赎回，保持配置平衡

### 减仓操作步骤

1. **确定赎回总额**：根据资金需求确定
2. **计算各基金赎回金额**：

```
单只基金赎回金额 = 总赎回金额 × (该基金当前市值 ÷ 组合总市值)
```

3. **检查持有期限**：

- 优先从持有超过1年的基金中赎回
- 如果某基金持有不足1年份额不够，从其他持有超1年基金多赎

4. **执行赎回**：

- 在交易日下午3点前提交申请
- 注意资金到账时间（通常T+3工作日）

### 临时周转方案

**建议设立应急资金层**：

1. **第一层**：货币基金（1-2个月生活费）
2. **第二层**：短期债券基金（2-6个月生活费）
3. **第三层**：投资组合（长期资金）

## 五、再平衡策略（优化版）

再平衡的核心目的是**“高抛低吸”**与**“控制风险”**，而非追求极致的精准。为了减少频繁交易带来的摩擦成本（手续费、精力），本策略采用“宽阈值 + 季度体检”的双重机制。

### 1. 再平衡触发矩阵

| 再平衡类型         | 检查频率                  | 触发阈值 (绝对值)     | 逻辑说明                                                                                                   |
| :----------------- | :------------------------ | :-------------------- | :--------------------------------------------------------------------------------------------------------- |
| **常规阈值再平衡** | **每月第一个交易日**      | **偏离目标比例 ±5%**  | 适用于所有资产类别。例如：目标20%，当前>25%或<15%时触发。这是最主要的调节手段。                            |
| **定期体检再平衡** | **每季度末** (3/6/9/12月) | **无条件检查**        | 即使未触发±5%，若季度末偏离度超过 **±3%**，也进行微调。防止资产在“阈值边缘”长期漂移。                      |
| **极端熔断再平衡** | **实时监控** (盘中/收盘)  | **偏离目标比例 ±10%** | 针对黑天鹅事件。一旦任一资产暴涨/暴跌导致偏离超过10%（如黄金从10%涨到20%），**立即执行**，不等待月度检查。 |
| **内部结构再平衡** | **仅在季度体检时检查**    | **比例偏离 > 50%**    | 仅在股票内部(300/500)或债券内部(国开/国债)比例严重失衡时调整。平时不做内部倒仓。                           |

### 2. 详细判断标准与操作示例

#### A. 常规与定期再平衡 (月度/季度)

- **判断标准**：
  - 计算公式：$|当前比例 - 目标比例| \geq 5\%$
  - _注：对于目标比例较小的资产（如10%的商品），±5%相当于波动了50%，这是一个非常安全的阈值。_
- **操作示例**：
  - 假设组合100万。
  - **场景**：沪深300（目标20%）大涨，当前市值26万（26%）；中证500（目标10%）下跌，当前市值8万（8%）。
  - **分析**：沪深300偏离+6%（触发），中证500偏离-2%（未触发）。
  - **操作**：卖出超配的沪深300（卖出6万元市值，使其回归20%），**将卖出的6万元全部买入偏离度最大的低配资产（中证500）**。
  - _修正逻辑：优先填最大的坑，而不是撒胡椒面。_

#### B. 内部结构再平衡 (季度)

- **判断标准**：
  - **股票内部**：沪深300与中证500的市值比。若比例超过 **3:1** 或低于 **1:1**（原定2:1），则调整。
  - **债券内部**：长债与中债的市值比。若比例超过 **2:1** 或低于 **1:2**（原定1:1），则调整。
- **操作示例**：
  - **场景**：季度检查发现，沪深300市值18万，中证500市值12万（目标各15万）。
  - **分析**：当前比例1.5:1，未超过3:1的阈值。
  - **操作**：**不操作**。尽管有偏离，但在允许范围内，保留动量。
  - _修改理由：内部倒仓不产生新的风险敞口变化，频繁操作意义不大，反而增加手续费。_

#### C. 极端情况处理 (熔断)

- **触发条件**：任一资产单日或单周波动剧烈，导致其仓位比例瞬间突破 **目标比例 ±10%**。
- **操作示例**：
  - **场景**：黄金目标10%，突发地缘政治危机，黄金一周暴涨，持仓占比瞬间达到21%。
  - **操作**：**立即卖出**11%仓位的黄金（即总资产的11%），将资金转入货币基金或低配资产，无需等待月底。

### 3. 再平衡执行流程 (SOP)

1. **估值**：每月1号（或季度末）统计所有场外基金的当前持有市值。
2. **计算**：计算各资产当前占比，并与目标占比做差。
3. **筛选**：
   - 找出所有 $|偏离度| \geq 5\%$ 的资产（红色警报）。
   - 若无红色警报，但在季度末，找出 $|偏离度| \geq 3\%$ 的资产（黄色警报）。
4. **执行**：
   - **卖出**：将所有“红色/黄色警报”的超配部分卖出（使其回归目标比例）。
   - **买入**：将回笼资金，按**偏离度从大到小**的顺序，优先补足低配资产，直至资金用完或所有资产回归目标范围。
5. **记录**：在Excel或记账软件中记录本次再平衡的日期、原因及操作金额。

```
赎回金额 = 组合总市值 × (当前比例 - 目标比例) [当当前比例>目标时]
申购金额 = 组合总市值 × (目标比例 - 当前比例) [当当前比例<目标时]
```

4. **执行交易**：

- 赎回超配基金（优先赎回持有>1年份额）
- 申购低配基金（按目标比例分配申购金额）

5. **记录归档**：记录再平衡日期、原因、操作详情

## 六、费用优化建议

### 申购费优惠渠道

- 天天基金、蚂蚁财富：通常申购费1折
- 基金公司官网/APP：可能有更低费率
- 设置定投计划：部分平台免申购费

### 赎回费节省技巧

| 持有时间 | 赎回费率 | 优化建议       |
| -------- | -------- | -------------- |
| <7天     | 1.5%     | 避免短期赎回   |
| 7天-1年  | 0.5%     | 尽量持有超1年  |
| >1年     | 0%       | 优先赎回这部分 |

### 长期持有成本

- 预计年化总成本：0.35%-0.45%
- 主要构成：管理费+托管费+申购费分摊

## 八、重要提醒

### 投资前须知

1. **资金性质**：用3-5年不用的闲钱投资
2. **风险认知**：任何投资都有风险，历史表现不代表未来
3. **长期坚持**：全天候策略需坚持3年以上才能体现效果
4. **纪律执行**：严格按照策略执行，避免情绪化操作

### 特别注意事项

1. **国开债波动**：7-10年久期债券对利率敏感，4%阈值有助于及时控制风险
2. **QDII基金特性**：国泰大宗商品为QDII基金，净值公布较晚，注意申购赎回时间
3. **税务考虑**：国开债利息收入需缴税，通过基金投资由管理人处理税务
4. **应急准备**：始终保持3-6个月生活费的流动性资产，避免被迫在不利时机赎回投资组合

### 常见问题处理

1. **基金暂停申购**：用表格中的替代基金或暂时投入货币基金
2. **大额资金需求**：提前1-2个月开始分批赎回
3. **市场极端波动**：坚持再平衡纪律，这正是策略价值所在
4. **平台操作问题**：熟悉交易平台功能，设置价格提醒和定投计划

### 关键时间节点备忘

- **季度末**：定期再平衡检查
- **每月初**：阈值检查
- **每年1月**：年度复盘与策略评估
- **持有满1年**：检查是否有可免税赎回的份额
