# 动量趋势双重过滤轮动策略

## 核心逻辑

**强者恒强 + 趋势过滤 + 震荡防御**，依托 12 只全赛道 ETF，每月双周筛选短期动量最强标的，结合均线确认趋势，通过分级止损和震荡过滤控制回撤，不主观择时、不预测行情，机械执行即可实现长期稳健增强收益。

| 支柱     | 机制                        | 作用                                       |
| -------- | --------------------------- | ------------------------------------------ |
| 强者恒强 | 选取近期动量排名前 N 的标的 | 捕捉行业轮动中的相对强势品种               |
| 趋势过滤 | 收盘 > MA短 且 MA长上行     | 排除均线死叉或价格跌破均线的品种，顺势而为 |
| 震荡防御 | 分级止损 + 震荡市过滤       | 控制回撤，避免在横盘震荡中频繁换仓磨损     |

## 复盘 / 调仓时间

### 1. 调仓复盘时间

**每两周的周四**执行一次复盘与调仓。

## 选股规则

### 前期准备

1. 统计 12 只轮动标的近 20 日涨幅，由高到低排序，作为动量评分依据。
2. 查看 12 只基金对应指数的收盘价与 20 日均线、60 日均线的位置关系，判断各标的趋势状态。
3. 查看沪深300指数（510300）收盘价与 60 日均线的关系，确定当前市场模式：
   - 收盘价 **> 60 日均线**，或 **≥ 80% 的标的收盘价站在 20 日均线上方** → **进攻模式**，正常执行轮动选股
   - 收盘价 **≤ 60 日均线** → **防守模式**，需调整持仓策略

### 操作规则

1. **进攻模式**：12 只标的全部纳入候选，正常执行动量轮动选股。
2. **防守模式**：仅可从以下 4 只防御性标的中选择买入：沪深300（006131）、中证500（006382）、红利（007467）、黄金（000217）。

## 买入条件

### 筛选条件（四条须同时满足）

1. 过去 20 个交易日涨跌幅排名前 2
2. 过去 20 个交易日涨幅 **≥ 3%**（绝对动量门槛，过滤弱势标的）
3. 收盘价站上 20 日均线
4. 60 日均线向上（中长期趋势向好，避免逆势买入）

### 具体操作

| 情形     | 满足条件的标的数              | 操作                             |
| -------- | ----------------------------- | -------------------------------- |
| 全额买入 | 2 只                          | 等金额买入，每只仓位约 50%       |
| 部分买入 | 1 只                          | 买入该只，仓位 50%，剩余资金闲置 |
| 空仓等待 | 0 只，或仅 1 只但不愿单独持有 | 保持空仓，等待下次复盘           |

## 卖出条件

任意一个触发，立即执行。

### 1. 趋势破位

- 持仓 ETF 连续 2 日收盘价有效跌破 20 日均线 → **立即减仓至 15%**，减仓资金转入货币基金
- 若价格继续跌破 60 日均线 → **清仓**

### 2. 排名过滤

- 持仓 ETF 在 12 只标的近 20 日涨幅排名中跌出**前 6 名** → **立即卖出该 ETF**
  - 找到满足买入条件的替代标的 → 直接换仓
  - 未找到替代标的 → 卖出资金转入货币基金

### 3. 仓位控制

- 单只 ETF 因涨幅导致仓位超过 50% → **立即卖出超额部分**
  - 另一只持仓标的仍满足条件 → 超额资金补入另一只，维持两只各 50%
  - 无合适标的 → 超额资金转入货币基金

### 4. 止损强制减仓

- 单只 ETF 持仓亏损超过 **10%** → **强制减仓 50%**，无论其他条件是否触发

## 注意事项

### 1. 标的维护

- 每季度末检查一次标的池：
  - 若某只 ETF 规模低于 **50 亿**（008021 除外）或跟踪误差过大，及时替换为同类别更优标的
  - 货币基金若出现收益大幅下滑或规模缩水，可替换为同类型其他货币基金

### 2. 备选标的使用规则

1. **使用场景限定**：备选标的仅用于"季度标的维护"或"标的紧急替换"（如现有标的出现清盘风险、跟踪误差过大），**不纳入日常轮动筛选范围**，避免标的过多增加筛选难度。

2. **同类别替换原则**：替换时须严格遵循同类别对应关系，不跨类别替换，保证标的池风格稳定：

   | 被替换标的类别       | 可用备选标的                         |
   | -------------------- | ------------------------------------ |
   | 科技类（AI、半导体） | 021085 光伏、017033 机器人（科技类） |
   | 防御类               | 007136 新能源（防御类）              |
   | 货币基金             | 003003 备选货币基金                  |

3. **替换后管理**：替换完成后，原标的从标的池中移除，新替换的备选标的正式纳入日常轮动筛选，后续按季度维护规则定期检查。

### 3. 机械执行

- 严格遵守所有规则，**不主观判断、不临时更改策略**
- 不因市场涨跌随意加仓、减仓或换标的
- 震荡市不焦虑，牛市不贪多
- 空仓期间坚持持有货币基金，**不盲目抄底**

### 4. 收益复盘

- 每年年底复盘一次全年收益，包含：
  - 轮动收益
  - 空仓期间货币基金收益
- 对比策略年化收益、最大回撤与沪深300指数
- 若出现大幅偏差，**仅优化震荡过滤、止损阈值，不改变核心逻辑**
